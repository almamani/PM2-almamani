Sobre el Proyecto

El proyecto consiste en una Web con temática de Cine. Cuenta con tres secciones: Principal, Nosotros, Te Recomendamos y Crear Película. Utilizamos las tecnologías necesarias para plasmar un listado de películas obtenidas de una Base de Datos, y un Formulario para cargar dichas películas.

Permite el acceso a Nuestro Catálogo, muestra una serie de películas a fin de que el cliente pueda seleccionar la película que quiera visualizar. Estas Películas se obtienen de una Base de Datos, si no se puede acceder al servidor que las contiene, se muestra una versión local.

Incluimos un resúmen de la información referida a la diseñadora de la web a cargo del sitio, detallamos lo más importante del proyecto, describiendo de que trata cada sección. Finalmente listamos las tecnologías utilizadas y la aplicación que efectuamos de cada una.

Esta sección se modificará cada semana con los detalles de la película recomendada, incluirá el motivo de dicha recomendación, además del trailer e información relacionada a la película que se destacó.

Consiste en un formulario que contará con los campos (Title, Year, Director, Duration, Genre, Rate, Poster) que permite la carga de Películas en la Base de Datos.

Tecnologías

Permite el maquetado de cada una de las páginas del Sitio Web, y el linkeado que permite pasar de sección en sección.

A través de CSS definimos estilos propios con lo que pudimos asignar colores personalizados y el formato diseñado.

Define el comportamiento, las acciones que se realizan al cargar el sitio y luego de la interacción con el usuario.

Framework del que tomamos componenetes como el acordeon y el carrusel, y algunos aspectos de formato

Permite ejecutar Javascript del lado del servidor para poder definir el backend del Sitio y el trabajo con los datos.

Mongo DB nos permite manipular la Base de Datos para la carga y consulta de la información de las películas.
